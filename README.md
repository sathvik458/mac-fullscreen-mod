# Mac Full Screen Scale

A tiny Chrome extension I am building to scale 16:9 videos
(YouTube, Netflix, JioHotstar) so they fill my Mac display.

Mac laptops have a 16:10 screen, but most videos are 16:9.
That leaves black bars on top and bottom. This extension zooms
the video just enough to kill the bars.
